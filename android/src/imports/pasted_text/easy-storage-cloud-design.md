Refined AI Prompt (Technical + Finder-Style UI)

Design a premium, technical user interface for an application called Easy Storage Cloud.

The app converts an external storage device connected to an Android phone into a remotely accessible cloud drive. The phone acts as a gateway server and exposes the storage via a relay tunnel to a web interface.

Create a professional, elegant UI similar to the aesthetics of
Nextcloud and the file management clarity of Finder.

The design should feel like a high-end operating system utility or developer tool, not a generic startup dashboard.

Visual Style

Focus on precision, hierarchy, and subtle elegance.

Design principles:

minimal but highly structured layout

clean separation between panels

subtle depth and shadows

smooth micro-animations

strong typography hierarchy

calm dark technical theme

Avoid heavy neon effects or flashy gradients.
